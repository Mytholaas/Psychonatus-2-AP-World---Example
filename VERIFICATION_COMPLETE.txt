================================================================================
                         TROUBLESHOOTING
================================================================================

Q: Console doesn't appear when I press ~
A: Check that UE4SS-settings.ini has:
   ConsoleEnabled = 1
   GuiConsoleVisible = 1

Q: Extraction happens but no files created
A: Check the console for error messages. Verify the output path in main.lua
   matches: C:\Users\adkin\source\repos\Psychonauts-2-AP-World---Example\extracted_data\

Q: Game crashes during extraction
A: This is rare but can happen. Try:
   - Extract from smaller areas first
   - Restart game between extractions
   - Make sure you're not in a cutscene or menu

Q: Some items are missing from extraction
A: Some items only spawn after story events. Try:
   - Progressing further in the story
   - Returning to areas later
   - Checking if items were already collected

Q: Python script errors
A: Make sure to save the Python files from the IDE first!
   The scripts are open in Visual Studio but need to be saved.
