────────────────────────────────────────────────────────────────────────────────
EXECUTIVE SUMMARY
────────────────────────────────────────────────────────────────────────────────

The Psychonauts 2 Archipelago randomizer requires walkable floor paths in the
Collective Unconscious (CU) area connecting the spawn point to all 13 mental
world doors. Without these paths, players cannot access mental worlds when
Mental Connection is randomized to a late-game location, creating softlocks.

The Python randomizer has been verified as correctly configured. The UE4 mod
needs to implement the floor system described in this specification.

PROBLEM STATEMENT
────────────────────────────────────────────────────────────────────────────────

Current State:
  • Collective Unconscious lacks complete walkable floor coverage
  • Some mental world doors require Mental Connection to reach (via floating)
  • Python randomizer is designed for Mental Connection to be optional

Result Without Fix:
  • Player spawns in CU with no Mental Connection
  • Cannot reach certain mental world doors
  • Cannot progress in game
  • SOFTLOCK - seed is unwinnable

────────────────────────────────────────────────────────────────────────────────
REQUIRED SOLUTION
────────────────────────────────────────────────────────────────────────────────

Implement walkable floor paths in Collective Unconscious connecting:
  • Spawn point
  • All 13 mental world doors

Doors that must be accessible:
  1. Loboto's Labyrinth           8. Cruller's Correspondence
  2. Hollis' Classroom            9. Tomb of the Sharkophagus
  3. Hollis' Hot Streak          10. Bob's Bottles
  4. Compton's Cookoff           11. Cassie's Collection
  5. Psi King's Sensorium        12. Lucrecia's Lament
  6. Ford's Follicles            13. Fatherland Follies
  7. Strike City

────────────────────────────────────────────────────────────────────────────────
RECOMMENDED IMPLEMENTATION: INVISIBLE FLOOR MESHES
────────────────────────────────────────────────────────────────────────────────

Approach:
  1. Hook Collective Unconscious level load event
  2. Get positions of all 13 mental world door actors
  3. Spawn invisible static mesh actors with collision
  4. Create paths connecting spawn to each door
  5. Add safety net floor below to catch falling players

Benefits:
  • Natural gameplay feel
  • Preserves intended game experience
  • No modification to player controls
  • Invisible floors maintain visual aesthetics

Technical Details: See UE4_MOD_SPECIFICATION.txt

────────────────────────────────────────────────────────────────────────────────
ALTERNATIVE IMPLEMENTATION: FLIGHT MODE IN CU
────────────────────────────────────────────────────────────────────────────────

Approach:
  1. Detect when player enters Collective Unconscious
  2. Enable flight mode or reduce gravity
  3. Restore normal movement when leaving CU

Benefits:
  • Simpler to implement
  • No need to identify door positions
  • Automatically works if doors move

Drawbacks:
  • Less natural gameplay feel
  • May enable unintended sequence breaks

Technical Details: See UE4_MOD_SPECIFICATION.txt
