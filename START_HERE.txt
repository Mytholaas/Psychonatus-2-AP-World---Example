# 🎮 Psychonauts 2 Archipelago - Player Package

## 🚀 Quick Start (3 Steps!)

### 1️⃣ Install the World
**Easiest Method**: Double-click `psychonauts2.apworld`
- It will automatically install to Archipelago
- Done!

**Alternative**: Run `INSTALL.bat`
- Automated installation for Windows
- Copies everything to the right places

### 2️⃣ Configure Your Settings
1. Open `Psychonauts2_Player.yaml` in any text editor
2. Change `name: Player1` to your name
3. Pick your win condition (normal, all_bosses, all_scav_hunt, scav_hunt_and_maligula)
4. Pick your starting outfit
5. Save the file

### 3️⃣ Generate Your Seed
1. Copy your edited YAML to Archipelago's `Players` folder
2. Run ArchipelagoGenerate.exe
3. Play!
