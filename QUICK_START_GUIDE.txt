## Troubleshooting

**UE4SS Console Not Showing**:
- Press ~ (tilde) or  (backtick) key
- Check UE4SS.log in the game folder

**No Data Extracted**:
- Make sure you're in an actual level (not main menu)
- Wait a few seconds after loading before pressing CTRL+F1
- Check the console for error messages

**Game Crashes**:
- Try extracting from smaller areas first
- Restart game between extractions if needed
