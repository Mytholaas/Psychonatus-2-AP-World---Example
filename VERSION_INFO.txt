INSTALLATION:

QUICK METHOD:
1. Double-click psychonauts2.apworld (auto-installs)
2. Edit Psychonauts2_Player.yaml
3. Generate your seed!

AUTOMATED METHOD:
1. Run INSTALL.bat
2. Follow the prompts
3. Done!

MANUAL METHOD:
1. Copy psychonauts2.apworld to [Archipelago]\lib\worlds\
2. Copy Psychonauts2_Player.yaml to [Archipelago]\Players\
3. Edit the YAML file
4. Generate your seed!

See README_PLAYERS.txt for detailed instructions.
