================================================================================
               QUICK GUIDE: GRULOVIA EXTRACTION SOLUTION
================================================================================

YOU ASKED: How can I extract Grulovia data without being able to access it?

ANSWER: Use the new GLOBAL EXTRACTION feature!

STEPS:
------
1. Launch Psychonauts 2
2. Load ANY save file (doesn't need to be complete)
3. Wait for main menu/game to load
4. Press: CTRL + F3
5. Done! Check: extracted_data\global_extraction.txt

WHAT IT DOES:
-------------
- Extracts ALL collectibles from ALL levels at once
- Includes Grulovia even if you can't access it
- Works by searching ALL game objects in memory
- Faster than visiting each area individually

OUTPUT FILE:
------------
global_extraction.txt will contain:
- Summary by collectible type
- Breakdown by level (including Grulovia!)
- Full object paths for each item
- Coordinates where available

VERIFICATION:
-------------
To confirm Grulovia was extracted:
1. Open global_extraction.txt
2. Search for "Grulovia" 
3. You should see collectibles listed under that level

NEW KEYBINDINGS:
----------------
CTRL + F1 = Extract current area only
CTRL + F2 = Dump all classes (debug)
CTRL + F3 = GLOBAL extraction (ALL levels)

RECOMMENDATION:
---------------
Use CTRL+F3 (global extraction) instead of visiting areas individually.
It's faster and gets Grulovia data automatically!
